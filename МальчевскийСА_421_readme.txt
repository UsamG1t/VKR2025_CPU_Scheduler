Программная реализация состоит из файлов:

 - algo1.py - Описание класса Algo1
 - algo2.py - Описание класса Algo2
 - objects.py - Описание классов Task, Queue_item, CPU
 - __main__.py - Основной код программы, включая генератор исходных данных
 - heatmap_percent.py - Код для визуализации результатов

 Шаги для запуска алгоритмов:
 1. Проверить наличие библиотек fractions, copy, itertools, random, glob, pandas, matplotlib, seaborn на устройстве, с которого проводится запуск, обеспечить на устройстве поддержку python3.13;
 2. Создать на уровне с файлами кода директории "VKR_results" и "VKR_Graphics"
 3. Запустить основной код программы: `python3.13 __main__.py`. В директории "VKR_results" будет сохранён набор csv-файлов с численными результатами экспериментов;
 4. Запустить код для визуализации результатов: `python3.13 heatmap_percent.py`. В директории "VKR_Graphics" будет сохранён набор png-изображений с визуализацией экспериментов.
